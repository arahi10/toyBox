## やり残したこと
高速化

  - 扱うデータは全て整数な分、classdefも含めてnumpy.ndarrayに書き換えが原理的には可能です。numbaのjitとか使えそう
  - もう少し枝刈りの方法を賢くやりたい、とは思う

  - 解の実行可能性に重きを置いた分、トレードオフとして下位の評価値の有効性がややない
